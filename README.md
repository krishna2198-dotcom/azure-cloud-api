# Azure Cloud API 🚀
A Flask REST API deployed on Microsoft Azure App Service.

## Live URL
https://krish-cloud-api-01-ceakcdg9h3anb0ge.westus2-01.azurewebsites.net

## Tech Stack
- Python 3.10
- Flask
- Microsoft Azure App Service
- Application Insights (Monitoring)

## How to Run Locally
pip install flask
python app.py
